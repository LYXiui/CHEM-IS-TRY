CHEM-IS-TRY 程式功能說明
========================

【專案簡介】
CHEM-IS-TRY 為網頁版虛擬化學實驗台，使用 React + Vite 建置，
標題僅顯示「CHEM-IS-TRY」，介面採遊戲化深色實驗室風格。

【啟動方式】
1. 開啟專案資料夾
2. 執行：npm install
3. 執行：npm run dev
4. 瀏覽器開啟 http://localhost:5173

【主要區塊】
1. 週期表
   - 標準週期表排列（18 欄、鑭系／錒系列）
   - 左鍵點元素：設為「選取中的元素」並置於實驗台（不必先放燒杯）
   - 右鍵：顯示元素簡介

2. 實驗台
   - 週期表選元素即可開始（如燃燒實驗）；傾倒溶液需先放燒杯
   - 上方：溶液與試劑櫃（右鍵簡介）
   - 火柴：燃燒實驗 · 酒精燈：加熱混合實驗
   - 點選物質 →「移除選取」刪除
   - 長反應可碼表加速

3. 合成圖鑑
   - 依「選取中的元素」列出可能解鎖的化合物
   - 右鍵卡片可看簡介

4. 實驗筆記
   - 僅在實驗成功且有化學變化時記載實驗式
   - 點實驗式展開過程；展開後可重現實驗

【資料檔】
src/data/elements.js    — 118 元素
src/data/compounds.js   — 試劑與化合物
src/data/reactions.js   — 反應表
src/data/periodicLayout.js — 週期表座標
src/data/equipment.js   — 實驗器具說明

【筆記本】
- 手寫風格；僅成功且有變化時記載實驗式
- 點實驗式展開過程；可重現實驗

【動畫】
火焰、變色、沉澱下落、冒泡、攪拌、過濾、分層、
元素沉入／浮出、指示劑逐漸變色

【文件】
docs/更新紀錄.txt          — 版本變更
docs/化學反應類型與範例.txt — 反應分類例題

【GitHub】
見 README.md：.gitignore、LICENSE、Pages 部署
