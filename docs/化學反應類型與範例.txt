CHEM-IS-TRY — 化學反應類型與範例（含網頁模擬對照）
========================================================
操作共通：火柴＝燃燒；酒精燈＝加熱（needsHeat 反應）；溶液須先放燒杯。
驗證指令：npm run test:reactions（檢查 reactions.js 鍵是否可觸發）

【1. 化合反應（合成）】
定義：兩種或多種物質生成一種新物質。
範例：
  2H₂ + O₂ → 2H₂O
  2Mg + O₂ → 2MgO
  C + O₂ → CO₂
  H₂ + Cl₂ → 2HCl
網頁模擬（✓ 已驗證）：
  · 週期表選 H、O →「混合」→ 得水（亦可先點火柴再「燃燒」）
  · Mg + O → 點火柴 →「燃燒」→ 耀眼白光、MgO
  · C + O → 火柴 →「燃燒」→ CO₂
  · H + Cl →「混合」→ HCl

【2. 分解反應】
定義：一種化合物分解成兩種或多種物質。
範例：
  2H₂O → 2H₂ + O₂
  2KClO₃ → 2KCl + 3O₂
  CaCO₃ → CaO + CO₂
網頁模擬：
  · ✓ 試劑架「蒸餾水」入燒杯 → 點酒精燈 →「混合」→ 簡化為 H₂ + 想像框完整式
  · ✓ 試劑架「碳酸鈣」單獨入燒杯 → 酒精燈 →「混合」→ CO₂（高溫分解簡化）
  · △ 氯酸鉀加熱：未內建試劑，文件保留教科範例

【3. 置換反應】
定義：較活潑元素把較不活潑元素從化合物中置換出來。
範例：
  Zn + H₂SO₄ → ZnSO₄ + H₂↑
  Fe + CuSO₄ → FeSO₄ + Cu
  Cu + 2AgNO₃ → Cu(NO₃)₂ + 2Ag
網頁模擬（✓）：
  · 週期表 Zn + 試劑架鹽酸或稀硫酸 →「混合」→ 冒 H₂
  · Fe + 硫酸銅溶液 → 藍液變淡綠（產物簡化為 FeCl₂(aq) 模型）
  · Cu + 硝酸銀溶液 → 藍色溶液、沉澱動畫

【4. 複分解反應（離子交換）】
定義：離子互相交換，常伴隨沉澱、氣體或水。
範例：
  AgNO₃ + NaCl → AgCl↓ + NaNO₃
  BaCl₂ + H₂SO₄ → BaSO₄↓ + 2HCl
  NaOH + HCl → NaCl + H₂O
  CaCO₃ + 2HCl → CaCl₂ + H₂O + CO₂↑
網頁模擬（✓，試劑架兩兩混合）：
  硝酸銀 + 氯化鈉溶液；氯化鋇 + 稀硫酸；
  氫氧化鈉 + 鹽酸；碳酸鈣 + 鹽酸（冒泡）

【5. 氧化還原反應】
定義：電子轉移。
範例：Cu + 濃 HNO₃；KMnO₄ + H₂O₂（催化／氧化）
網頁模擬（✓）：
  · Cu + 硝酸 → 藍綠液、氣體想像框
  · 過錳酸鉀 + 過氧化氫 → 紫褪色、冒泡

【6. 酸鹼中和反應】
定義：酸與鹼生成鹽和水。
範例：HCl + NaOH；H₂SO₄ + 2NaOH
網頁模擬（✓）：
  · 氫氧化鈉 + 鹽酸（或稀硫酸）→ 放熱／蒸氣
  · 可另加酚酞或石蕊於燒杯，遇鹼有呈色動畫（reactionAnim.js）

【7. 燃燒反應】
定義：可燃物與氧氣劇烈氧化。
範例：CH₄ + 2O₂ → CO₂ + 2H₂O；乙醇燃燒
網頁模擬（✓，需火柴，不需酒精燈）：
  · 先 C+H「混合」解鎖甲烷 → 燒杯放入 CH₄ + 週期表 O → 火柴 →「燃燒」
  · 先 C+H+O「混合」得乙醇 → 乙醇 + O → 火柴 →「燃燒」
  · Mg+O、C+O 亦屬燃燒類

【8. 沉澱反應】
定義：難溶固體析出。
範例：AgCl（白）、BaSO₄（白）、Cu(OH)₂（藍）
網頁模擬（✓）：
  · AgNO₃ + NaCl；BaCl₂ + 稀硫酸 → 白色沉澱動畫
  · NaOH + 硫酸銅 → 藍色 Cu(OH)₂ 沉澱

【9. 氣體生成反應】
定義：產生氣體冒泡。
範例：Zn + 2HCl；CaCO₃ + 2HCl
網頁模擬（✓）：
  · Zn + 鹽酸；碳酸鈣 + 鹽酸 → bubble／gas 動畫

【10. 催化反應】
定義：催化劑改變速率但不消耗。
範例：2H₂O₂ →(MnO₂) 2H₂O + O₂↑
網頁模擬（✓）：
  · 過錳酸鉀溶液 + 過氧化氫 → 與【5】同一鍵，紫色褪色、冒泡
