CHEM-IS-TRY — 專題演示講稿與實驗示範
========================================
建議總時長：12～15 分鐘（簡報 6～8 分＋現場演示 5～7 分）
啟動：npm run dev → http://localhost:5173
演示前：瀏覽器全螢幕（F11）、可先「清空」實驗台

────────────────────────────────────────
一、開場（約 30 秒）
────────────────────────────────────────

各位老師、同學好，我是 ________。今天要介紹的專題是 **CHEM-IS-TRY**——
一個用網頁程式設計技術打造的**虛擬化學實驗台**。

本專案是單頁應用程式（SPA），使用 React、Vite 建置，
整合週期表、試劑櫃、實驗動畫與實驗筆記，並涵蓋 **118 種元素**的反應資料。

（切換至網頁畫面）

────────────────────────────────────────
二、介面導覽（約 1 分鐘，邊講邊指）
────────────────────────────────────────

畫面上方是**實驗台**：
  · 實驗器具列：燒杯、酒精燈、火柴、攪拌棒等
  · 中央工作區：放置元素與溶液、觀看反應動畫
  · 下方左側：**溶液與試劑櫃**（可點「展開」看全部試劑）
  · 下方右側：**元素週期表**（118 元素，點選即可實驗）

右側欄：
  · **合成圖鑑**：依選取元素提示可解鎖的化合物
  · **實驗筆記本**：成功反應會自動記錄，可展開過程並**重現實驗**

操作口訣（可念給聽眾）：
  「固態元素直接點；液體溶液先放燒杯；燃燒點火柴；加熱開酒精燈。」

右鍵任一元素、試劑或器具，可跳出**簡介說明**。

────────────────────────────────────────

每組演示後可短停 3 秒，讓聽眾看「實驗報告」白底區與動畫。

────────────────────────────────────────
四、加碼示範（時間充裕時選 1～2 項）
────────────────────────────────────────

【A】加熱分解 — 碳酸鈣（分解反應＋酒精燈）
  清空 → 燒杯 → 傾倒 CaCO₃ → 點酒精燈 → 混合
  講稿：「需加熱的反應要開酒精燈，模擬高溫分解產生二氧化碳。」

【B】氧化還原 — 過錳酸鉀＋過氧化氫
  燒杯 → KMnO₄(aq) + H₂O₂(aq) → 混合
  講稿：「紫色溶液褪色並冒泡，展示氧化還原與催化。」

【C】118 元素覆蓋 — 鋰與水（資料庫生成反應）
  週期表點 Li → 燒杯 + H₂O → 混合
  講稿：「系統依資料庫為 118 元素建立反應路徑，稀有元素也能模擬。」

【D】實驗筆記重現
  完成任一示範後 → 右側筆記本點最新實驗式 → 展開過程 →「重現此實驗」
  講稿：「筆記存在 localStorage，可回顧步驟並一鍵重播。」

【E】合成圖鑑
  週期表點 Cu → 右側圖鑑顯示可合成物 → 右鍵看簡介
  講稿：「圖鑑依選取元素動態提示，引導探索合成路徑。」

────────────────────────────────────────
五、技術亮點串詞（搭配簡報第 2～17 頁，約 2 分鐘）
────────────────────────────────────────

「在技術面，我們用 **React 元件** 拆分週期表、實驗台與筆記本；
  **useState / useEffect** 管理實驗狀態與 localStorage 持久化；
  **reactions.js** 以查表法驅動反應，手動設計反應與 118 元素自動生成反應合併；
  **CSS 動畫** 呈現火焰、沉澱、氣泡等現象；
  **Vite** 負責開發與打包，**Tailwind** 輔助響應式排版。
  驗證方面可執行 npm run test:all 檢查化學式、反應鍵與元素覆蓋。」
